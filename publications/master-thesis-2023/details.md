# Design and Realization of a Wireless Crack Detection System Based on IoT and Acoustic Emission

## Thesis Information
**Type:** Master Thesis  
**University:** University of Calabria  
**Department:** Department of Computer Engineering, Modeling, Electronics and Systems Engineering  
**Date:** October 2023

## Supervisors
- Prof. Francesco Lamonaca

## Abstract

This master thesis presents a comprehensive study on the design and implementation of a wireless crack detection system that combines Internet of Things (IoT) technologies with Acoustic Emission (AE) sensors for real-time structural health monitoring. The research addresses the critical need for efficient and reliable crack detection in infrastructure, which is essential for preventing catastrophic failures and ensuring public safety.

## Research Objectives

The primary goals of this research include:
- Design a cost-effective wireless sensor network for crack detection
- Integrate Acoustic Emission technology for high-precision monitoring
- Develop IoT-based data transmission and cloud storage solutions
- Create a real-time visualization and alerting system
- Validate the system through extensive laboratory and field testing

## System Architecture

### Hardware Components
- **Acoustic Emission Sensors**: High-sensitivity transducers for detecting crack propagation
- **Signal Processing Circuit**: Custom-designed amplification and filtering circuitry
- **Microcontroller**: ESP32-based processing unit with WiFi/Bluetooth capabilities
- **GPS Module**: Microsecond-resolution timestamping for event correlation
- **Power Management**: Optimized for long-term deployment

### Software Components
- **Embedded Firmware**: Real-time signal processing and event detection
- **MQTT Protocol**: Reliable data transmission to cloud servers
- **Node-RED Dashboard**: Real-time visualization and monitoring interface
- **Cloud Database**: Scalable storage for historical data analysis

## Key Contributions

1. **Multi-node Support**: Scalable architecture supporting multiple distributed sensors
2. **Real-time Processing**: Microsecond-level event detection and timestamping
3. **Cloud Integration**: Seamless data flow from sensors to cloud infrastructure
4. **User-friendly Interface**: Intuitive dashboard for monitoring and analysis
5. **Validated Performance**: Successful testing in laboratory and real-world conditions

## Methodology

The research methodology encompassed:
- Literature review of crack detection techniques
- Hardware design and prototyping
- Software development and integration
- Laboratory validation with controlled crack propagation
- Field testing in real infrastructure
- Performance analysis and optimization

## Results and Findings

The developed system demonstrated:
- High accuracy in detecting crack initiation and propagation
- Reliable wireless communication over extended distances
- Real-time data transmission with minimal latency
- Robust performance in various environmental conditions
- Cost-effectiveness compared to traditional monitoring systems

## Applications

The system is applicable to:
- Bridge and building structural monitoring
- Pipeline integrity assessment
- Railway infrastructure monitoring
- Industrial equipment condition monitoring
- Historical structure preservation
