Custom Android 15 Tablet

A fully functional custom tablet built from scratch using AOSP (Android Open Source Project) and Raspberry Pi hardware. This project demonstrates deep expertise in embedded systems and Android internals, from bootloader configuration to system service optimization.

Technical Implementation:

  • Built Android 15 from AOSP source code with custom configurations
  • Integrated hardware abstraction layers (HALs) for Raspberry Pi
  • Implemented touchscreen driver support and input system optimization
  • Custom bootloader configuration and system initialization
  • Performance tuning and system service optimization
  • Display driver integration and graphics acceleration setup

Key Components:

  • AOSP (Android Open Source Project) 15
  • Raspberry Pi hardware platform
  • Custom touchscreen integration
  • Modified Android HAL for hardware support
  • Optimized system services and performance

Automotive Embedded System

A comprehensive automotive embedded system solution that combines the reliability of STM32MP1 with the advanced capabilities of i.MX93 processors. This unified system provides a robust platform for modern vehicle applications, offering both real-time processing and advanced features.

Key features include:

  • Two versions of the system: STM32MP1 and i.MX93
  • Real-time processing capabilities for critical vehicle functions
  • Automotive-grade hardware integration and safety features
  • Enhanced security features for automotive applications
  • 4G LTE cellular connectivity for remote monitoring
  • GPS module for precise location tracking
  • Ethernet for high-speed data transfer
  • USB interfaces for peripheral expansion
  • WiFi for wireless connectivity
  • Bluetooth Low Energy (BLE) for short-range communication
  • Advanced analog-to-digital conversion (ADC) for sensor integration
